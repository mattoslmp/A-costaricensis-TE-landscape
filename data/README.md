# Source tables

- `A_costaricensis_TE_families.csv` contains the confirmed transposable-element families used to generate the repeat landscape.
- `A_costaricensis_TE_families_full.csv` contains the complete exported table, including confirmed TEs and unsupported unknown repeats.

The main repository README provides a command to reconstruct the Excel workbook expected by `scripts/plot_te_landscape.py`.
